2.3 Hardware Requirements

The hardware development boards supported by Harmony 3 Smart Energy applications are as follows:

  • PIC32CXMTG-EK

  • PIC32CXMTSH-DB

  • PIC32CXMTC-DB

  • ATSAMD20-XPRO

Connected to:

  • PL460-EK for PLC communications

  • ATREB215-XPRO-A for RF communications

The Demo Application built and tested on this Getting Started guide uses a set of (PIC32CXMTG-EK + PL460-EK + ATREB215-XPRO-A) boards.

Refer to the User guide of these development boards for more details.
